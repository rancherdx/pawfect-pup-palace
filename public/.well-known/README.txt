Apple Pay Domain Verification

Place the official file provided by Apple/Square here with the exact filename:

  apple-developer-merchantid-domain-association

This path is served as https://<your-domain>/.well-known/apple-developer-merchantid-domain-association

Note: The Admin > Secure Integrations section lets you save a reference copy to Supabase Storage, but Apple will only validate the file at this path in your deployed site.
