
# Puppy Breeder App

A full-stack application for puppy breeders built with Cloudflare Workers, D1 Database, KV Store, and R2 Storage.

## Features

- Authentication with JWT
- Puppy and Litter management
- Image uploads for puppies
- Admin dashboard for breeders
- Square payments integration (coming soon)
